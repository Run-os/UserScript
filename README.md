## 🚀 安装方法

1. 安装浏览器扩展管理器：
   - [Tampermonkey](https://www.tampermonkey.net/) (推荐)
   - [Violentmonkey](https://violentmonkey.github.io/)
   - [Greasemonkey](https://www.greasespot.net/) (仅Firefox)

2. 点击脚本文件链接，或复制脚本代码到扩展管理器中安装

3. 访问对应网站即可自动运行脚本
