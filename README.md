# Runos/UserScript

个人油猴（Tampermonkey）脚本集合。

## 目录结构（按用途分层）

顶层目录按脚本的**用途**划分，不再按「在用 / 失效」做状态分层；失效脚本按其用途归入对应目录，并在下方表格的「状态」列标注。

- `工作/` —— 工作相关（内网数据剔除等）
- `考试/` —— 考试 / 学习相关（保密教育自动播放、学法网自动答题、题库 API 文档）
- `工具/` —— 通用浏览器增强小工具（百度网盘字幕等）

```
UserScript/
├── LICENSE
├── README.md
├── 工作/
│   └── 数据剔除.user.js
├── 考试/
│   ├── 保密教育自动点击播放.user.js
│   ├── 学法网-自建题库版.user.js
│   ├── 学法网-全能题库版.user.js      # 已失效 / 旧版，仅供存档参考
│   └── 自建题库API文档.md
└── 工具/
    └── 百度网盘打开中文字幕.user.js
```

## 命名规范

- 统一以 `.user.js` 结尾，便于 Tampermonkey 直接安装与更新检测。
- 文件名与脚本 `@name` 保持一致，避免安装后管理混乱（例如 `保密教育自动播放.js` → `保密教育自动点击播放.user.js`，对齐其 `@name`）。

## 脚本一览

| 文件 | `@name` | 说明 | 适用站点 | 状态 | 路径 |
| --- | --- | --- | --- | --- | --- |
| 数据剔除.user.js | 数据剔除 | 自动创建控制面板、填充 / 递增日期 | 工作内网 `85.16.18.180:8080` | 在用 | `工作/数据剔除.user.js` |
| 保密教育自动点击播放.user.js | 保密教育自动点击播放 | 自动点击播放保密教育视频 | `www.baomi.org.cn` | 在用 | `考试/保密教育自动点击播放.user.js` |
| 学法网-自建题库版.user.js | 湖南学法网12348自动答题--自建题库版 | 自动答题并上传自建题库 | `hn.12348.gov.cn` | 在用 | `考试/学法网-自建题库版.user.js` |
| 学法网-全能题库版.user.js | 湖南学法网12348自动答题--全能题库版 | 自动答题（旧版 / 全能题库），已被「自建题库版」取代 | `hn.12348.gov.cn` | 已失效 | `考试/学法网-全能题库版.user.js` |

| 自建题库API文档.md | — | 自建题库 API 接口说明 | — | 文档 | `考试/自建题库API文档.md` |

> 注：学法网两个脚本的文件名使用简洁版（`学法网-xxx版`），其 `@name` 仍含站点全称「湖南学法网12348自动答题--」。如需把 `@name` 也同步为简洁版以保持完全一致，可告知我修改（注意：修改 `@name` 会让已安装的 Tampermonkey 脚本被视为「新脚本」）。
